# server-github-pull
EC2 서버에서 Github의 키를 생성하고 등록한 후 `git pull`을 호출하고 서버를 다시 재기동 하는 프로그램 작성

## Feature
- Restart server
- Github source code pull
- ~~Generate RSA key~~
- Reigter Github url through cli
